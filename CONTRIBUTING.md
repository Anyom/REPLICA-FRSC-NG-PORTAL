# Contributing to FRSC Portal

Thank you for your interest in contributing to the FRSC Nigeria Portal! Here's how you can help:

## Getting Started

1. **Fork the repository**
2. **Clone your fork**: `git clone https://github.com/your-username/REPLICA-FRSC-NG-PORTAL.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make your changes**
5. **Commit**: `git commit -m 'Add feature'`
6. **Push**: `git push origin feature/your-feature`
7. **Create a Pull Request**

## Code Style

- Use TypeScript for type safety
- Follow ESLint rules
- Use Prettier for formatting
- Write meaningful commit messages

## Development Workflow

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run linting
npm run lint

# Type check
npm run type-check

# Format code
npm run format
```

## Areas to Contribute

- 🐛 Bug fixes
- ✨ New features
- 📚 Documentation
- 🎨 UI/UX improvements
- ⚡ Performance optimizations
- ♿ Accessibility improvements
- 🌐 Internationalization

## Pull Request Process

1. Ensure tests pass
2. Update documentation if needed
3. Add meaningful commit messages
4. Reference relevant issues
5. Wait for review
6. Address feedback

## Code of Conduct

- Be respectful
- Be inclusive
- Report issues responsibly
- Follow project guidelines
